# Example Prompts

Use these prompts to try Feishu Workspace quickly.

## 1. Meeting notes → Feishu Doc

Summarize this discussion into clean meeting notes and write them to our Feishu doc.

## 2. Wiki lookup → concise summary

Search the Feishu wiki for onboarding docs and give me a concise summary.

## 3. Action items → Bitable

Turn this conversation into action items and add them to our Bitable with owner, due date, and status.

## 4. Standup notes → project doc

Append today's standup notes to the project doc in Feishu.

## 5. Planning discussion → structured brief

Create a project brief in Feishu based on this chat, with goals, scope, risks, and next steps.
